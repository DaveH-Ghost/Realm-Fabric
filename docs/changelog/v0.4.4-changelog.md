# Version 0.4.4 Changelog

**Purpose:** Living record of **V0.4.4** slices.

**Baseline:** Builds on **V0.4.3** (`v0.4.3`) — memory module UI, edit location, witness broadcast.

**Overall V0.4.4 focus:** **Prompt and schema token reduction** — shrink default compound-turn input from ~966 est. tokens to create headroom before future lorebook and richer memory systems. Session save/load deferred to **V0.4.5**.

See also: [ROADMAP.md](../ROADMAP.md), [v0.4.3-changelog.md](v0.4.3-changelog.md).

## 0.4.4b — Compact prompt (non-breaking)

**Status:** ✅ **Done**

- Rewrote `_compound_turn_rules()` — terse bullets, no markdown bold, no repeated char-limit prose
- Trimmed `look_and_interact_block()` — dynamic lists only; removed emote example paragraph and speak footer
- Compact `compound_output_format()` — minimal JSON skeleton + one-line field glossary
- Removed duplicate glue text block before `output_format` in `default_prompt_blocks()`
- Shorter character slot labels (`Personality:` / `Description:`)

GM-editable verbose rules remain available via **Prompt layout** session overrides.
